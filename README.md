# Нейросеть для предсказания стоимости жилья

Простая нейронная сеть на PyTorch для предсказания цен на недвижимость.

## Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Обучите модель:
```bash
python housing_model.py
```

3. Используйте модель для предсказаний:
```bash
python predict.py
```

## Архитектура

- Входной слой: 8 признаков
- Скрытые слои: 64 → 32 → 16 нейронов
- Выходной слой: 1 нейрон (цена)
- Активация: ReLU
- Регуляризация: Dropout (0.2)

## Признаки датасета California Housing

1. MedInc - медианный доход в районе
2. HouseAge - возраст дома
3. AveRooms - среднее количество комнат
4. AveBedrms - среднее количество спален
5. Population - население района
6. AveOccup - среднее количество жильцов
7. Latitude - широта
8. Longitude - долгота

## Производительность

Модель обучается за ~2-3 минуты на CPU и достигает приемлемой точности для демонстрации.
